Introduction to Kotlin

Kotlin is a modern programming language developed by JetBrains that runs on the JVM. It's designed to be:

  • Concise and safe
  • Interoperable with Java
  • Tool-friendly
  • Modern and pragmatic
// Kotlin is modern and concise
fun main() {
    println("Hello, Kotlin!")
}

Setting Up Kotlin

To get started with Kotlin, you'll need:

  1. JDK (Java Development Kit) 8 or later
  2. IntelliJ IDEA (recommended) or Android Studio
  3. Kotlin plugin (usually pre-installed)
// Create a new Kotlin project in IntelliJ IDEA
// File -> New -> Project -> Kotlin/JVM

Creating a New Project in IntelliJ IDEA

Follow these steps to create a new Kotlin project:

  1. Open IntelliJ IDEA
  2. Click on "File" → "New" → "Project"
  3. Select "Kotlin/JVM" from the left sidebar
  4. Configure your project:
    • Project name: Your project name
    • Project location: Choose your directory
    • Language: Kotlin
    • Build system: Gradle (recommended)
    • JDK: Select your installed JDK
  5. Click "Create"

Understanding Android Packages

Android packages are essential for organizing your code:

package com.example.myapp

// Import Android packages

// MainActivity class

Common Android Packages

  • android.os - Core Android OS functionality
  • android.widget - UI components
  • androidx.appcompat - App compatibility library
  • androidx.core - Core Android extensions
  • androidx.lifecycle - Lifecycle components
  • androidx.room - Database components
  • androidx.navigation - Navigation components

Coroutines

Kotlin coroutines for asynchronous programming:

import kotlinx.coroutines.*

// Basic coroutine
fun main() = runBlocking {
    launch {
        delay(1000)
        println("World!")
    }
    println("Hello")
}

// Async/await
suspend fun fetchData(): String {
    delay(1000)
    return "Data"
}

// Coroutine scope
class MyViewModel {
    private val viewModelScope = CoroutineScope(Dispatchers.Main)
    
    fun loadData() {
        viewModelScope.launch {
            val result = async { fetchData() }.await()
            // Handle result
        }
    }
}

Variables and Data Types

Kotlin has two types of variables:

// Mutable variable (var)
var mutableVar = "Can be changed"

// Immutable variable (val)
val immutableVar = "Cannot be changed"

// Type inference
val number = 42 // Int
val text = "Hello" // String
val decimal = 3.14 // Double
val isTrue = true // Boolean

// Explicit type declaration
val explicitInt: Int = 42
val explicitString: String = "Hello"

Functions

Functions in Kotlin are declared using the 'fun' keyword:

// Basic function
fun greet(name: String): String {
    return "Hello, $name!"
}

// Single-expression function
fun add(a: Int, b: Int) = a + b

// Function with default parameters
fun greet(name: String, greeting: String = "Hello") {
    println("$greeting, $name!")
}

// Function with named parameters
fun createUser(
    name: String,
    age: Int,
    email: String
) {
    // Function body
}

Control Flow

Kotlin provides various control flow structures:

// If expression
val max = if (a > b) a else b

// When expression (similar to switch)
when (x) {
    1 -> print("x is 1")
    2 -> print("x is 2")
    else -> print("x is neither 1 nor 2")
}

// For loop
for (i in 1..5) {
    println(i)
}

// While loop
while (condition) {
    // Loop body
}

// Do-while loop
do {
    // Loop body
} while (condition)

// Object (Singleton)

        // Connection logic
    }
}

Null Safety

Kotlin's null safety features help prevent NullPointerException:

// Non-nullable type
var nonNullString: String = "Hello" // Cannot be null

// Nullable type
var nullableString: String? = null // Can be null

// Safe call operator
val length = nullableString?.length

// Elvis operator
val name = nullableString ?: "Guest"

// Not-null assertion
val length2 = nullableString!!.length // Throws exception if null

Collections

Kotlin provides powerful collection types:

// List
val list = listOf(1, 2, 3)
val mutableList = mutableListOf(1, 2, 3)

// Set
val set = setOf(1, 2, 3)
val mutableSet = mutableSetOf(1, 2, 3)

// Map
val map = mapOf("key" to "value")
val mutableMap = mutableMapOf("key" to "value")

// Collection operations
val numbers = listOf(1, 2, 3, 4, 5)
val doubled = numbers.map { it * 2 }
val evenNumbers = numbers.filter { it % 2 == 0 }

2. Todo List App

data class Todo(
    val id: Int,
    var title: String,
    var completed: Boolean = false
)

class TodoList {
    private val todos = mutableListOf()
    
    fun addTodo(title: String) {
        todos.add(Todo(todos.size + 1, title))
    }
    
    fun completeTodo(id: Int) {
        todos.find { it.id == id }?.completed = true
    }
    
    fun getTodos() = todos.toList()
}

fun main() {
    val todoList = TodoList()
    todoList.addTodo("Learn Kotlin")
    todoList.addTodo("Build Android App")
    todoList.completeTodo(1)
    println(todoList.getTodos())
}
